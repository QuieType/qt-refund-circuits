# Crosscode Refund Circuits Mod

Were you ever annoyed in crosscode whn you wanted to test out combat arts and you realized that spending cp was permanent? 

The Crosscode speedrunners felt the same, but, unlike us *casual* gamers, speedrunners do things about what they don't like. [EpicYoshiMaster](https://www.youtube.com/watch?v=Qcyp5n-xKR8), a speedrunner, decided to make a [mod](https://github.com/CCDirectLink/cc-open-circuits) to fix that, meaning that us non-speedrunners could use it too. This was great for a lot of us, but kinda annoying for me, who found myself often in the position of recommending mods to new players. I wouldn't recommend Yoshi's mod, since it included a 'cheat' mode where all circuits were freely unlocked, and I it's not quite condusive to have cheat options in the menus when you're running your first playthrough.

With Yoshi's kind permission, I copied half her mod's code into this one, making a mod that only provides *some* of the functionality instead of all of it. I also added a single line to the cutscene at the end of Temple Mine where you recieve Circuit Reset items for the first time, saying, and I paraphrase, *Here you go, but these are actually useless.*

You can and should install this mod through the standard modloader, to install which Xenon has made an excellent [video guide](https://www.youtube.com/watch?v=7oBYnF_xMvU) for. If you have any reason to install it manually, you either also know how to or are currently being told to by someone else, who you should also ask how.
